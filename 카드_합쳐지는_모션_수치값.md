# 한번에 열기 - 카드 합쳐지는 모션 수치값 정리

## 1. 카드 합쳐지는 애니메이션 (Merge)

### 기본 설정
- **Transition 시간**: `0.4s`
- **Easing 함수**: `cubic-bezier(0.4, 0, 0.2, 1)`
- **대기 시간**: `500ms` (합쳐지는 애니메이션 완료 대기)

### Transform 값
- **기본 위치**: `translate(-50%, -50%) translateX(0) translateY(0) rotate(0deg) rotateY(0deg)`
- **최종 위치**: `finalTargetTransform` (맨 앞 카드의 현재 위치 기준)

### Z-index 설정
- **맨 앞 카드**: `3 + currentCardIndex` (예: 3장이면 6)
- **뒤 카드들**: `3 + (currentCardIndex - i)` (예: 2번째 카드는 4, 1번째 카드는 3)

---

## 2. 바운스 애니메이션 (Bounce)

### 애니메이션 설정
- **지속 시간**: `1s`
- **반복 횟수**: `2번` (총 2초)
- **Easing 함수**: `cubic-bezier(0.45, 0, 0.55, 1)`
- **Animation Fill Mode**: `both`

### Scale 값
- **시작/끝**: `scale(1)`
- **중간 (50%)**: `scale(1.025)` (2.5% 확대)

### Transform 값
- **기본**: `translate(-50%, -50%) translateX(0) translateY(0) rotate(0deg) rotateY(0deg) scale(1)`
- **중간**: `translate(-50%, -50%) translateX(0) translateY(0) rotate(0deg) rotateY(0deg) scale(1.025)`

### 대기 시간
- **바운스 애니메이션 대기**: `2000ms` (1s × 2번)

---

## 3. 보더라인 애니메이션 (Border Loading)

### 애니메이션 설정
- **지속 시간**: `1.5s`
- **반복**: `infinite` (무한 반복)
- **Easing 함수**: `linear`

### 각도 변화
- **시작**: `0deg`
- **끝**: `360deg`

### 그라데이션 설정
- **시작 각도**: `transparent 0deg`
- **중간**: `transparent 60deg`
- **하이라이트 시작**: `rgba(255, 255, 255, 0.9) 90deg`
- **하이라이트 최대**: `rgba(120, 200, 255, 1) 120deg`
- **하이라이트 끝**: `rgba(255, 255, 255, 0.9) 150deg`
- **끝**: `transparent 180deg ~ 360deg`

---

## 4. 스페셜 카드 페이드인 (일반 → 스페셜 전환)

### 페이드인 설정
- **지속 시간**: `0.6s`
- **Easing 함수**: `cubic-bezier(0.4, 0, 0.6, 1)` (앞부분 ease-in, 뒷부분 ease-out)
- **대기 시간**: `600ms` (페이드인 완료 대기)

### Transform 값
- **시작**: `scale(1.01)`
- **끝**: `scale(1)`

### Opacity 값
- **시작**: `0`
- **끝**: `1`

---

## 5. 카드 플립 애니메이션 (Flip)

### 플립 설정 (일반 카드)
- **지속 시간**: `0.7s`
- **Easing 함수**: `cubic-bezier(0.4, 0, 0.2, 1)`
- **Transition 지연**: `10ms` (애니메이션 재생을 위한 지연)

### 플립 설정 (스페셜 카드)
- **지속 시간**: `1.0s`
- **Easing 함수**: `cubic-bezier(0.34, 1.56, 0.64, 1)`

### Transform 값
- **시작**: `translate(-50%, -50%) translateX(0) translateY(0) rotate(0deg) rotateY(0deg)`
- **끝**: `translate(-50%, -50%) translateX(0) translateY(0) rotate(0deg) rotateY(180deg)`

### Z-index 설정
- **맨 앞 카드**: `10`
- **뒤 카드들**: `1`

### 대기 시간
- **플립 완료 대기**: `800ms`

---

## 6. 스페셜 카드가 맨 앞일 때 추가 대기 시간

### 안정화 시간
- **대기 시간**: `200ms` (바운스 애니메이션 종료 후)

---

## 전체 타이밍 순서

1. **카드 합쳐지기**: `0.4s` (transition)
2. **합쳐지는 애니메이션 완료 대기**: `500ms`
3. **바운스 애니메이션**: `2000ms` (1s × 2번)
4. **스페셜 카드가 맨 앞일 때**: `200ms` (안정화 시간)
5. **일반 → 스페셜 페이드인** (해당되는 경우): `600ms`
6. **플립 애니메이션**: `700ms` (일반) / `1000ms` (스페셜)
7. **플립 완료 대기**: `800ms`

---

## Easing 함수 정리

- **카드 합쳐지기**: `cubic-bezier(0.4, 0, 0.2, 1)` (ease-out)
- **바운스 애니메이션**: `cubic-bezier(0.45, 0, 0.55, 1)` (ease-in-out)
- **스페셜 페이드인**: `cubic-bezier(0.4, 0, 0.6, 1)` (ease-in-out, 앞부분 ease-in, 뒷부분 ease-out)
- **플립 (일반)**: `cubic-bezier(0.4, 0, 0.2, 1)` (ease-out)
- **플립 (스페셜)**: `cubic-bezier(0.34, 1.56, 0.64, 1)` (bounce 효과)

---
